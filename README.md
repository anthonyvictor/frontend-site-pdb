### Introdução
Site desenvolvido com `Next.js` no frontend e `Node.js` no backend

### Instalação
```bash
git clone https://github.com/pizzaria-delicia-da-bahia/backend.git
npm install

cd ..

git clone https://github.com/pizzaria-delicia-da-bahia/frontend.git
npm install
```
### Inicialização
```bash
cd backend
npm run dev

cd ..

cd frontend
npm run dev
```
